In jeder Runde treffen Sie eine Entscheidung darüber, in welcher Reihenfolge Sie sich an den Schulen bewerben möchten. Das heißt, Sie erstellen eine Präferenzliste über die Schulen. Sie müssen alle vier Schulen in der Präferenzliste auflisten. Sie und alle anderen Mitglieder Ihrer Gruppe schicken die Präferenzlisten an eine zentrale Zulassungsstelle.

Beispiel:



Sobald Sie und alle anderen Mitglieder in Ihrer Gruppe die Präferenzlisten an die zentrale Zulassungsstelle übermittelt haben, werden die Zulassungen ermittelt. Wie genau die Zulassungen ermittelt werden, wird auf den folgenden Seiten erklärt. Für dieses Beispiel gilt:

Sie erhalten 18 ECU, wenn Sie einen Platz an Schule A zugeteilt bekommen.

Sie erhalten 3 ECU, wenn Sie einen Platz an Schule B zugeteilt bekommen.

Sie erhalten 8 ECU, wenn Sie einen Platz an Schule C zugeteilt bekommen.

Sie erhalten 13 ECU, wenn Sie einen Platz an Schule D zugeteilt bekommen.
